# EmotionMap
EmotionMap — это веб-приложение для ежедневного отслеживания эмоционального состояния. Пользователи могут фиксировать настроение, оставлять заметки и анализировать динамику.

# Описание
Приложение позволяет создавать записи с оценкой от 1 до 5, текстом и категорией. Доступны календарь с цветовой индикацией, графики настроения, статистика, фильтрация и поиск.

# Функциональность
Регистрация и вход  
Создание, редактирование, удаление записей  
Календарь с подсветкой дней  
Статистика и графики  
Фильтрация по категориям  
Поиск по заметкам  
Редактирование профиля  

# Технологии
React  
Express  
MongoDB  
Vercel / Render
