# Treker-mentalky
MoodTracker — это веб-приложение для ежедневного отслеживания настроения. Пользователи оценивают настроение от 1 до 5, добавляют заметки и категории. Функции: регистрация, календарь с цветовой индикацией, статистика, графики настроения, поиск и редактирование профиля. Стек: React, Express, MongoDB
